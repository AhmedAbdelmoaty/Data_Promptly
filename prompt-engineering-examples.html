Prompt Engineering Examples & Mistakes

Learn from real-world examples of effective prompts and common mistakes to avoid.

Prompt Engineering Examples

Basic Prompt

"Explain machine learning"

This might produce a very general overview without focus on your specific needs. The response could be too technical for some audiences or too vague for others.

Intermediate Prompt

"Explain machine learning to a business executive in 3 paragraphs, focusing on practical applications rather than technical details. Include 2 real-world examples from the retail industry."

This will produce a targeted response with executive-level language, practical focus, and specific retail examples like recommendation systems and inventory forecasting, all in a concise format.

Advanced Prompt

"Act as a senior data scientist with 10 years of experience in retail. Explain how machine learning can improve inventory management to a skeptical store manager who is concerned about cost. Use analogies from sports to make it relatable, and include 3 specific examples of ML techniques that could be applied. Format the response as a persuasive email with subject line 'Unlocking Inventory Efficiency' and keep it under 400 words."

This generates a professionally formatted email with sports analogies comparing inventory to a basketball team's roster, specific techniques like demand forecasting with XGBoost, and cost-benefit analysis tailored to retail managers' concerns.

Common Prompt Engineering Mistakes

Being Too Vague

Prompts like "Tell me about AI" are too broad. The AI doesn't know what aspect you're interested in or at what level to respond.

Fix: Specify what you want to know and for what purpose. "Explain how AI is used in healthcare to improve patient outcomes, using 3 specific examples."

Assuming Context

Prompts that assume the AI knows what you're referring to, like "Continue the story" without providing the beginning.

Fix: Always provide necessary context. "Here's the beginning of a story about a detective in 1920s Chicago: [text]. Continue the story in the same style for 3 paragraphs."

Overly Complex Requests

Asking for too many things at once, like "Write a comprehensive guide to digital marketing covering SEO, PPC, content, and social media in 500 words."

Fix: Break into smaller prompts or specify you want an outline first. "Create an outline for a comprehensive guide to digital marketing, then we'll expand each section."

Ignoring Formatting

Not specifying how you want the output formatted, leading to responses that might not be immediately usable.

Fix: Specify format needs. "Provide the information as a bulleted list with brief explanations for each point."

No Error Handling

Not anticipating how the AI might misinterpret your prompt or go off track.

Fix: Add guardrails. "If any part of this request is unclear, ask for clarification before proceeding."

Neglecting Iteration

Expecting perfect results from the first prompt rather than refining based on initial outputs.

Fix: Treat prompt creation as an iterative process, building on what works from previous attempts.

Prompt Engineering for Different Use Cases

Content Creation

Example Prompt: "Write a 300-word blog post introduction about the benefits of remote work for tech companies. Use a friendly, conversational tone and include 2 statistics from reputable sources."

Code Generation

Example Prompt: "Write a Python function that takes a list of numbers and returns a dictionary with the mean, median, and mode. Include docstring documentation and 3 test cases."

Data Analysis

Example Prompt: "Analyze this dataset of monthly sales figures [provide data]. Identify 3 key trends and suggest actionable recommendations for improving Q4 sales. Present the findings in a bulleted executive summary."

Learning & Education

Example Prompt: "Explain how neural networks work to a high school student who knows basic algebra. Use an analogy about how humans recognize faces, and include a simple diagram description."
